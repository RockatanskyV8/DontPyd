# DontPyd
Leitura e edição de textos do dontpad para o terminal linux

## Leitura

```
pyd.py nome
```

```
pyd.py nome/nome2
```

## Edição

```
pyd.py edit nome
```
```
pyd.py edit nome/nome2
```

## Upload

```
pyd.py send:caminho/do/arquivo.txt nome
```
```
pyd.py send:caminho/do/arquivo.txt nome/nome2
```
